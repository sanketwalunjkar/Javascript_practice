Welcome to jQuery-Responsive-Full-Screen-Plugin Pages.

With websites now showing their images as fullscreen backgrounds, We have compiled a jQuery code which will help you with full screen backgrounds with responsive solutions. We have deisgned this with support to cycle slider plugin. The advantage of this plugin is that it fixes the image on full screen from all the four sides rather than from any two sides.

                    imageWidth:         1600,

                    horizontalSpace:    0,
                    verticalPosition:   'center',
                    horizontalPosition:   'left'
imageHeight

Allows you to fix the image height

imageWidth

Allows you to fix the image width

verticalSpace

Allows you to manage vertical space.

horizontalSpace

Allows you to manage horizontal space.

verticalPosition

Allows you to manage the image vertically.

horizontalPosition

Allows you to manage the image horizontally.
