Useful Underscore Methods

Main Types

1)array functions 2)utility functions 3)collection functions 4)common functions 5)objects functions 6)Chaining 7)

  • (1)underscore.map

    The map function is used to iterate over an array and transform that array into another array. the function gets applied to each item in that array. The map function creates a new array from the return value of the iterator function. Anytime you need to modify an array into a new one by applying a function to each element in the array, map is the function to reach for.

    With Array

    let result = _.map([1,2,3,4,5], (number, key) => { return number * 3 ; });

    result => [3, 6, 9]

    With Object

    let resultObject = _.map({one: 1, two: 2, three: 3}, (num, key) => { return num + 2; });

    result => [5, 8, 11]

  • (2)underscore.find

    NOTE :: return only one value if(true){return element matching the criteria} else {return undefined}

    With Array(for true condition)

    let result = _.find([1, 2, 3, 4, 5, 6], (num) => { return num % 2 == 0; });

    result => 2

    With Array(for false condition)

    let result = _.find([1, 2, 3, 4, 5, 6], (num) => { return num > 6 });

    result => undefined

    With Objects

    let result = _.find([{name: 'vishal',age: 20},{name:'parth', age: 23},{name: 'rao', age: 24}], (person, index) => { return person.age < 24})

    result => {name: "vishal", age: "20"}

  • (3)underscore.filter

    _.filter return all sets of value matching the condition unlike _.find

    it gives the output in the format of the datatype of list passed ,i.e if (true) { result is array/array of objects} else { empty array}

    With Array(for true condition)

    let result = _.filter([1, 2, 3, 4, 5, 6], (num) => { return num % 2 == 0; });

    result => [2, 4, 6]

    With Array(for false condition)

    let result = _.filter([1, 2, 3, 4, 5, 6], (num) => { return num > 6 });

    result => empty array

    With Objects((for true condition))

    let result = _.filter([{name: 'vishal',age: 20},{name:'parth', age: 23},{name: 'rao', age: 24}], (person, index) => { return person.age < 24});

    result => [{name: "vishal", age: "20"}, {name:'parth', age: 23}]

    With Objects((for false condition))

    let result = _.filter([{name: 'vishal',age: 20},{name:'parth', age: 23},{name: 'rao', age: 24}], (person, index) => { return person.age > 24});

    result => []

  • (4)underscore.reject

    it is the exact opposite of _.filter method it returns all the valus that are not matching the given condition

    let result = _.reject([1, 2, 3, 4, 5, 6], (num) => { return num % 2 == 0; });

    result => [1, 3, 5]

  • (5)underscore.some

    it returns true if Returns true if any of the values in the list pass the predicate truth test. else returns false

    let result = _.some([0, 2, 3, 4, 5, 6], (num) => { return num > 0; });

    result => true

    let result = _.some([1, 3, 5], (num) => { return num % 2 == 0; });

    result => false

  • (6)underscore.every

    (strict than _.some method)it returns true if all the values in list are matching the given condition else returns false

    let result = _.every([1, 2, 3, 4, 5, 6], (num) => { return num > 0; });

    result => true

    let result = _.every([1, 2, 3, 4, 5, 6], (num) => { return num % 2 == 0; });

    result => false

  • (1)underscore.keys

    it gives all the keys of an object in one array

    let result = _.keys({name: 'vishal', age: 24});

    result => ['name', 'age']

  • (2)underscore.values

    it gives all the values of an object in one array

    let result = _.values({name: 'vishal', age: 24});

    result => ['vishal', 24]

  • (3)underscore.mapObject

    its just like array.map but its used for object

    _.mapObject({name: 'vishal', age: 24}, (val, key) => { return val + 5; });

    result => {name: "vishal5", age: 29}

  • (4)underscore.pairs

    it converts object into list of [key value] pairs.result is array

    let result = _.pairs({name: 'vishal', age: 24});

    result => [['name': 'vishal'], ['age': 24]]

  • (5)underscore.pick

    it picks the key value pairs based on specified condition

    let result = _.pick({name: 'vishal', age: 24} , (val, key) => { return val > 23 });

    result => {age: 24}

  • (6)underscore.omit

    .exact opposite of _.pick. it omits the key value pairs based on specified condition and return other key value pairs

    let result = _.omit({name: 'vishal', age: 24} , (val, key) => { return val > 23 });

    result => {name: 'vishal'}

  • (7)underscore.has

    returns true if given key is present in object else false

    let result = _.has({name: 'vishal', age: 24} , 'age');

    result => true

    let result = _.has({name: 'vishal', age: 24} , 'ages');

    result => false
